
A simple **Tic Tac Toe game** built using **Python Tkinter**.

---

## Features
- GUI with large, readable buttons
- Player 1 & Player 2 name input
- X/O choice selection
- Win/Draw detection
- Restart functionality (player names retained)
- Clean and interactive interface



## How to Run
1. Make sure **Python** is installed on your system.
2. Download the `tic_tac_toe.py` file.
3. Run the file using terminal or IDE:

```bash
python main.py
Enter Player 1 and Player 2 names, choose X or O, and enjoy the game!
